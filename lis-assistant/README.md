# CNMI LIS Assistant

เครื่องมือช่วยลดการคัดลอกข้อมูลทีละช่อง เมื่อตัว LIS ไม่มีระบบ Import

## วิธีใช้แบบปลอดภัยที่สุด

1. ในแอปเลือกวันที่ แล้วกด **ส่งออก LIS CSV**
2. ดับเบิลคลิก `Start-LIS-Assistant.cmd`
3. กด **เปิดไฟล์ CNMI-LIS-*.csv**
4. เลือกผู้บริจาค 1 ราย
5. ใช้ปุ่ม **คัดลอกช่องถัดไป** แล้ววางใน LIS
6. ตรวจข้อมูลก่อนบันทึกทุกครั้ง

## โหมด Auto TAB

โหมดนี้จะพิมพ์ค่าตามลำดับใน `lis-field-order.json` และกด TAB ระหว่างช่อง

ก่อนใช้จริงต้อง:

1. ทดลองกับข้อมูลจำลอง 1 ราย
2. ตรวจว่าลำดับ TAB ตรงกับ LIS จริง
3. แก้ `lis-field-order.json` ให้ตรงกับหน้าจอ LIS
4. ตรวจข้อมูลทุกช่องก่อนกดบันทึกใน LIS

เนื่องจากโนอายังไม่ได้เห็นหน้าจอและลำดับ Tab ของ LIS จริง จึงไม่ควรเปิด Auto TAB กับข้อมูลจริงทันที

ข้อมูลทั้งหมดทำงานภายในคอมพิวเตอร์ ไม่มีการส่งออกอินเทอร์เน็ต
