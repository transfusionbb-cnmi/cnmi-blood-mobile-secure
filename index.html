หนังสือรับรอง

ออกเลขและพิมพ์หนังสือรับรองการบริจาคโลหิต

เตรียมออกหน่วย / ส่งออก LIS

ตั้งช่วงเลขถุง นำเข้าประวัติผู้บริจาค ตรวจเครื่องอ่านบัตร และส่งออกข้อมูลกลับเข้าโรงพยาบาล

1. ตั้งค่างานออกหน่วย

ยังไม่ได้ตั้งช่วงเลขถุง

2. เครื่องอ่านบัตรประชาชน

เสียบเครื่อง uTrust 2700R แล้วเปิด smartcard-bridge/Start-Bridge.cmd บนคอมพิวเตอร์ Windows

ยังไม่ได้ตรวจสอบ

3. นำเข้าประวัติผู้บริจาค

รองรับไฟล์ Excel หรือ CSV โดยเลือกชีต Donors อัตโนมัติ ข้อมูลจะถูกเข้ารหัสและเก็บเฉพาะในเครื่องนี้

4. ส่งออกเพื่อกลับเข้า LIS

LIS ไม่มี Import จึงเตรียมทั้ง CSV และเครื่องมือช่วยกรอกทีละรายในโฟลเดอร์ lis-assistant

5. ปิดงานและล้างข้อมูลจากอุปกรณ์

ทำหลังตรวจสอบว่า Export และนำเข้าระบบหลักเรียบร้อยแล้วเท่านั้น
