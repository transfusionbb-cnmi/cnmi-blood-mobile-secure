# Smart Card Bridge — uTrust 2700R

ใช้สำหรับเชื่อมเว็บ `https://mobile.cnmiblood.com` กับเครื่องอ่านบัตรประชาชน uTrust 2700R บน Windows ผ่าน PC/SC โดยข้อมูลวิ่งเฉพาะในเครื่อง `127.0.0.1` และไม่ส่งขึ้น Cloud

## วิธีเปิด

1. เสียบ uTrust 2700R กับคอมพิวเตอร์ Windows
2. รอให้ Windows ติดตั้ง Driver และตรวจพบเครื่อง
3. ดับเบิลคลิก `Start-Bridge.cmd`
4. คงหน้าต่างสีดำไว้ระหว่างใช้งาน
5. เปิดแอป แล้วไปที่ **เตรียมออกหน่วย / ส่งออก LIS**
6. กด **ตรวจสอบเครื่องอ่านบัตร**
7. เสียบบัตร แล้วกด **อ่านบัตรประชาชน**

## ปิดระบบ

กด `Ctrl+C` หรือปิดหน้าต่าง Bridge หลังเสร็จงาน

## หมายเหตุ

- Bridge รับคำขอเฉพาะ `mobile.cnmiblood.com` และ localhost
- ไม่บันทึกเลขบัตรหรือข้อมูลผู้บริจาคเป็นไฟล์ Log
- ใช้ Windows Smart Card Service และมาตรฐาน PC/SC
- รุ่นแรกอ่านข้อมูลตัวอักษร ไม่อ่านรูปจากบัตร เพื่อให้เร็วขึ้น
